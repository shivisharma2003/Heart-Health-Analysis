# 🫀 Heart Health Analysis 2022 - Power BI Dashboard

## 📌 Project Overview
This project is a Data Analysis and Visualization dashboard built using **Microsoft Power BI**. The primary objective of this dashboard is to analyze patterns between patients' health histories, physical attributes (such as BMI, Height, and Weight), and overall heart health using a comprehensive 2022 dataset.

## 📂 Files Included in this Repository
1. **`heart health.pbix`**: The main Power BI Dashboard file containing all visualizations and data models.
2. **`heart health 2022.zip`**: Contains the `heart health 2022.csv` file, which includes detailed patient health histories (e.g., Smoker status, Alcohol consumption, Age category). *(Note: Due to the large file size of 75MB+, this dataset has been compressed into a ZIP file for upload).*
3. **`physical_attributes.csv`**: Contains physical data for the patients, including Height, Weight, and calculated BMI.

## 🚀 How to Run and View This Project
Since the main dataset is compressed, please follow these steps to ensure the dashboard loads correctly on your local machine:

**Step 1: Download the Repository**
* Click the green **`Code`** button at the top right of this page and select **`Download ZIP`**. (Alternatively, you can clone the repository).

**Step 2: Extract (Unzip) the Dataset**
* Once downloaded, extract the main repository folder.
* Navigate inside the folder, right-click the **`heart health 2022.zip`** file, and select **"Extract Here"** or **"Unzip"**. 
* *Crucial Step:* The `.csv` file must be extracted in the same directory for Power BI to load the data successfully.

**Step 3: Install Power BI Desktop**
* You need **Microsoft Power BI Desktop** installed on your system to view this file. If you don't have it, you can download it for free from the [official Microsoft website](https://powerbi.microsoft.com/desktop/).

**Step 4: Open the Dashboard**
* Double-click the **`heart health.pbix`** file.
* The file will open in Power BI, allowing you to interact with the dashboards, slicers, and visual insights!

## 💡 Key Insights & Tech Stack
* **Tool Used:** Microsoft Power BI
* **Data Processing:** Power Query (Data cleaning, merging, and transformation)
* **Features:** Interactive Slicers, Custom Visualizations, and Relational Data Modeling
